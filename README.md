# Wollok JS 

A JS environment for the [Wollok language](https://github.com/uqbar-project/wollok).

## Developing

Just run initially

```
yarn install
# or npm install
```

And then to run tests

```bash
yarn gulp test
# or npm test
```

To lint the code

```bash
yarn gulp lint
# or npm run lint
```
